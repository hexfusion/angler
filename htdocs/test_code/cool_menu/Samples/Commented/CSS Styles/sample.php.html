CSS Styles

COOLjsTree and COOLjsTree Professional allow you to have different CSS styles for each level. To specify style you can use following code:

var TREE1_FORMAT = [
    ...
//10. default style for all nodes
    "clsNode",
//11. styles for each level of tree
//(default style will be used for undefined levels)
    [
        "clsNodeLevel1",
        "clsNodeLevel2",
        "clsNodeLevel3",
        "clsNodeLevel4"
    ],
    ...

For level 1 style "clsNodeLevel1" will be used, for level 2 - "clsNodeLevel2", etc. For all other levels implicity will be used style "clsNode".
