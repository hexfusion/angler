You can have many more features!

Our Professional version offers many more advanced features. Here is the list with short descriptions:

Per node image customization

You can define separate set of images for individual nodes. I.e. one of the nodes can have special document icon, or larger button image (see demo and demo).

Explorer-like mode

Tree can be presented as connected set of nodes - just like in many applications with hierarchical views (see demo and demo):

Relative positioning

Relative positioning allow you to put the tree into container like <DIV> or <TD>. This means that tree will follow the flow of design: it can be centered, left or right aligned, and also it can resize its container when needed (see demo).

Full source code

Professional edition includes full uncompressed source code.

Special style for selected node

Selected (i.e. last clicked) node can have special CSS style and/or background color (see demo and demo).

Text wrapping

Tree nodes can contain many lines of text, and script will wrap them correctly (see demo):

Dynamic API

COOLjsTree Professional can be modified on the fly (at client's side) by calling its methods. You can add, modify, and remove nodes (see demo).

State in cookies

Tree can keep its state between pages. That means user will see tree with same expanded and selected nodes when he/she will go to another page with the same tree (see demo).
