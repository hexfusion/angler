1st Page message- 2nd link title (on the right) Insert text to show on "1st Page Author Message" as a 2nd link on bottom right of the message.
1st Page message - 2nd link tpl file (ex:portfolio.tpl) Insert the name of the file to link to the text inserted above.
1st Page - Columns per album page Set the number of columns to dysplay on first page, will not intervene with the number of collums displayed on other pages.

1st Page - Show Description on 1st page of the Gallery If checked, display the description for Gallery (base album) on 1st page, above the thumbnails.
1st Page - Show Information on 1st page of the Gallery If checked, display other information for Gallery (base album) on 1st page, above the thumbnails.
1st Page - Link Thumbnail directly to photo page (to work must activate "Image Block" Module) If checked, when clicking the thumbnail you will be redirected to the photo page overpassing the album page (uses the "Image Block" Module, this option will not work if this module isn't active).

Navigation - Show pics on the TOP of the Album If checked, the navigation buttons (first, previous, next, last) will be shown above the thumbnails.
Navigation - Show pics on the BOTTOM of the Album If checked, the navigation buttons (first, previous, next, last) will be shown below the thumbnails.

Show Information Display other information for the album.
Show Information above the Thumbnails after Title and Description (by default: side by side) If checked, display other information above the thumbnails, in line with top navigation buttons, else will be shown on left side of the thumbnails.

PHOTOHELP
Options to customize Navigation on Photo page
Navigation - Show pics ABOVE the Photo If checked, display navigation buttons (first, previous, next, last) above the photo.
Navigation - Show pics BELOW the Photo If checked, display navigation buttons (first, previous, next, last) below the photo.
Navigation - Show full size photo POPUP navigation pic If checked, display a "Full Size" button on navigation buttons (clicking open a pop up window showing the full size photo).
Navigation - Show POPUP navigation pic even if the current view is the full size image If checked, display the "Full Size" button on every photos, else the button will be shown only if the current view is not the full size image.
Navigation (OnMouseOver) - Show navigation pics ABOVE the Photo If checked, OnMouseOver the photo activate a javascript and display navigation buttons (first, next) on the top of the photo.
Navigation (OnMouseOver) - Show navigation pics BELOW the Photo If checked, OnMouseOver the photo activate a javascript and display navigation buttons (first, next) on the bottom of the photo.
Navigation (OnMouseOver) - Show full size photo POPUP navigation pic If checked, display a "Full Size" button on OnMouseOver the photo, clicking will open a pop up window width full size image.
Navigation (OnMouseOver) - Show POPUP navigation pic even if the current view is the full size image If checked, display a "Full Size" button on OnMouseOver the photo on every photo, else, the button will be shown only if the currente view is not the full size image.
Options to customize Photo page appearance
Show Title If checked, display the photo title.
Show Title ABOVE the Photo (by default: side by side) If checked, display the photo title above the photo, in line width top navigation buttons, else will be shown below micro thumbnails.
Show Description If checked, display the photo description.
Show Description ABOVE the Photo below the Title (by default: side by side) If checked, display the photo description above the photo, in line width top navigation buttons, else will be shown below micro thumbnails.
Show Information If checked, display photo other information.
Show Information above the Photo after Title and Description(by default: side by side) If checked, display photo other information above the photo, in line width top navigation buttons, else will be shown below micro thumbnails.
Show Size If checked, display photo size information.
Show Size even on full size image If checked, display photo size information on every image; else, if current view is full size image photo size will not be shown.
Clicking on full size link will show the full size image on a POPUP Window (by default: Tradicional full size link) If checked, clicking on full size link will open a pop up window with the full size image, else the ful size image will be shown on the same window.
Show POPUP link even if the current view is the full size image If checked, display photo full size link on every image; else, if current view is full size image the link will not be shown.
Show Photos fading in If checked, activate a javascript to display photos fadinf in.
Options to customize Micro Thumbnails appearance on photo page
MicroThumbs - Show MicroThumbs on the RIGHT of the image (by default: LEFT) If checked, display the navigation micro thumbnails on the right side of the photo, else will be shown on the left side.
MicroThumbs - Show always first and last MicroThumbs If checked, the first and the last micro thumbnails will be always shown.
MicroThumbs - Show navigation below MicroThumbs table, when they are not all visible If checked, if the micro thumbnails aren't all visible, below will be shown micro thumbnails navigation links (<<, >>) to display the other micro thumbnails.
MicroThumbs - Show first and last linked pics below MicroThumbs table, when they are not all visible If checked, the micro thumbnails navigation links will be substituted by buttons.
MicroThumbs - Set text to show width "first" and "last" pics or with "<<" ">>" (ex. : more) Insert text to display with micro thumbnails navigation links or buttons.
MicroThumbs - Set (peerWindowSize) number of MicroThumbs diplayed (plus first and last) Insert "peerWindowsSize", if the theme is set to show first and last MicroThumbs the number of micro thumbnails shown will float (will be the inserted number plus 0, 1 or 2, as either necessary to show the first or the last micro thumbnail), else, the number of micro thumbnails shown will be the inserted number.
MicroThumbs - Set number of columns per MicroThumbs Insert the number of micro thumbnails columns.
MicroThumbs - Set width for MicroThumbs Insert the width for the micro thumbnails.
MicroThumbs - Set opacity for MicroThumbs (1 to 10) Insert the opacity for the micro thumbnails (1 to 10, where 10 is 100% opacity).
MicroThumbs - Set opacity for MicroThumbs on MouseOver (1 to 10) Insert the opacity for the micro thumbnails OnMouseOver (1 to 10, where 10 is 100% opacity).
MicroThumbs - Set size increment for MicroThumbs on MouseOver Insert the number of pixel to incremet the size of micro thumbnails OnMouseOver.
BLOCKSHELP
Options to customize the way Blocks are shown on Album and Photo pages
Position the Blocks Table that displayed the selected Blocks to show on album and photo pages on the tradicional way (choices: left,center,right). Insert "left", "center" or "right" to set the position of Blocks Table - inside this table wil be displayed the blocks selected to be shown on album and on photo pages (EXIF, Comments,etc...).
Align the Blocks displayed inside the Blocks Table (choices: left,center,right). Insert "left", "center" or "right" to align Blocks inside the Blocks Table.
Show a "Blocks" button on photo pages (when clicked Blocks selected to show: containing EXIF, Comments,etc, will be displayed) If checked, on photo pages navigation, wil be shown a "Blocks" button, when clicking the blocks selected to display with photo pages will be shown on a pop up.

Album - display a "View Comments" button If checked, display a Album "View Comments" button on top navigation (it will display only if module is active and the user have permission to comments).
Options to customize Items Actions to display on album page
Items - Show Items actions below thumbnails on webmaster mode (for guests show a "Add to Cart" button if module is active) If checked, on webmaster mode display Items actions below thumbnails (on guest mode, display a "Add to Cart" button if this module is active).
Items - display "Comments" buttons If checked, display Items "Add Comments" and "View comments" buttons below thumbnails (it will display only if module is active and the user have permission to view/add comments).
Items - display "Show EXIF" button If checked, display a Item "EXIF" button below thumbnails (it will display only if module is active).
Items - display "Buttons" below Title, Description and Information. (by default: between Thumbnail and Title) If checked, display the Items action buttons below Title, Description and Information, else the buttons will be shown between the thumbnail and the title.
Options to customize Actions to display on photo page
Photo - Show "Photo" actions (on webmaster mode) If checked, on webmaster mode display Photo actions on the the photo navigation bar.
Photo - display a "Add Comments" button If checked, display a Photo "Add Comments" button on the photo navigation bar (it will display only if module is active and the user have permission to add comments).
If "Show a Blocks button on photo pages" is checked and "Show comments" is a selected Blocks to show on photo pages, only this button will be shown and the "Add Comments" button will not display.
Photo - display a "View Comments" button If checked, display a Photo "View Comments" button on the photo navigation bar (it will display only if module is active and the user have permission to view comments).
If "Show a Blocks button on photo pages" is checked and "Show comments" is a selected Blocks to show on photo pages, only this button will be shown and the "View Comments" button will not display.
Photo - display a "Add to Cart" button If checked, display a Photo "Add to Cart" button on the the photo navigation bar, only at PHOTO page (it will display only if module is active).
Photo - display a "Show EXIF" button If checked, display a Photo "Show EXIF" button on the photo navigation bar (it will display only if module is active).
If "Show a Blocks button on photo pages" is checked and "EXIF/IPTC photo info" is a selected Blocks to show on photo pages, only this button will be shown and the "Show EXIF" button will not display.
